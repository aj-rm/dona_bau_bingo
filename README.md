# Doña Bau 🎲

Una lotería personalizada creada para **Doña Bau**.

Este proyecto es una versión digital del tradicional juego de lotería, adaptada especialmente para compartir momentos divertidos en familia. El objetivo es conservar la esencia del juego clásico mientras se aprovecha la sencillez de una aplicación web.

## Características

* Interfaz simple y fácil de usar.
* Cartas de lotería personalizadas.
* Funciona directamente en el navegador.
* No requiere instalación ni dependencias externas.
* Compatible con dispositivos móviles y de escritorio.

## Tecnologías

* HTML
* CSS
* JavaScript

## Cómo ejecutar el proyecto

1. Clona este repositorio:

```bash
git clone <url-del-repositorio>
```

2. Abre el archivo `index.html` en tu navegador.

O simplemente visita la versión publicada del proyecto.

## Propósito

Este proyecto fue creado como un pequeño homenaje a Doña Bau y como una forma de compartir momentos en familia a través de la tecnología.
